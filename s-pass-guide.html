Understanding the S Pass

The S Pass is designed for mid-skilled foreign employees who want to work in Singapore. This pass category is suitable for technicians, skilled workers, and mid-level professionals across various industries.

Basic Requirements

Eligibility Criteria:
  • Minimum fixed monthly salary: S$3,150
  • Relevant qualifications and experience
  • Acceptable degree, diploma, or technical certification
  • Pass mark in points calculator system
Points Calculator Criteria:
  • Salary (20 points)
  • Educational qualifications (20 points)
  • Work experience (20 points)
  • Job type (20 points)
  • Additional skills (10 points)

Quota and Levy System

Sector-Specific Quotas:

Application Process

  1. Employer Application

    Employer submits application via EP Online

  2. Document Submission

    Upload required documents and certifications

  3. Assessment

    MOM evaluates application using points system

  4. Approval & Issuance

    If approved, complete formalities and collect pass

Required Documents

Personal Documents:
  • Valid passport
  • Educational certificates
  • Skills certificates
  • Recent passport photos
  • Resume/CV
Employment Documents:
  • Employment contract
  • Company's registration info
  • Detailed job description
  • Salary documents
  • Medical examination results

Employer Obligations

  • Pay monthly levy
  • Maintain quota limits
  • Provide medical insurance
  • Ensure safe working conditions
  • Comply with employment laws
  • Update MOM of changes

Common Challenges & Solutions

  • Quota limits - Plan workforce carefully
  • Salary requirements - Review compensation
  • Documentation - Use checklist provided
  • Points calculation - Seek professional help
  • Renewal issues - Start process early

Renewal Process

  • Apply 3 months before expiry
  • Update all required documents
  • Maintain qualifying salary
  • Check quota availability
  • Review levy payments
Quick Facts
  • Processing: 3 weeks
  • Validity: Up to 2 years
  • Min Salary: S$3,150
  • Quota: Sector-based
  • Levy: From $330
